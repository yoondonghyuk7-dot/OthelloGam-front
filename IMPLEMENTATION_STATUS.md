## 🌐 외부 접속 설정 (ngrok 사용)

### 학교 네트워크 환경
- 사설 IP = 공인 IP (포트포워딩 불필요)
- 중앙 방화벽 차단으로 외부 접속 불가
- 해결 방법: ngrok 또는 전산원 허용 요청

### ngrok 사용 방법

#### 1. ngrok 설치 및 계정 생성
```bash
# 1. https://ngrok.com/download 에서 다운로드
# 2. ngrok.com에서 무료 계정 생성
# 3. 인증 토큰 받기
ngrok config add-authtoken YOUR_AUTH_TOKEN
```

#### 2. 서버 실행
```bash
# 서버를 포트 80에서 실행
java -cp target/classes org.example.server.ServerLauncher
```

#### 3. ngrok 터널 생성
```bash
# 포트 80을 외부에 노출 (TCP 프로토콜)
ngrok tcp 80
```

#### 4. ngrok URL 확인
터미널에 다음과 같이 표시됩니다:
```
Forwarding: tcp://0.tcp.ngrok.io:12345 -> localhost:80
```
→ 호스트: `0.tcp.ngrok.io`
→ 포트: `12345`

#### 5. 클라이언트 설정
`config.properties` 파일 수정:
```properties
server.ip=0.tcp.ngrok.io
server.port=12345
```

**주의사항:**
- ngrok 무료 버전은 재시작 시마다 URL이 변경됩니다
- 매번 새로운 URL을 클라이언트에 배포해야 합니다
- ngrok 유료 버전은 고정 URL 사용 가능
